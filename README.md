# GigHub API

GigHub API is a FastAPI backend for managing freelance gigs.

*Features:*

- Create, read, update, delete gigs

- Search and filter gigs by category, budget, status

- Built-in data validation with Pydantic

- Auto-generated Swagger UI docs

*Admission Number:* C027-01-0543/2024

*How to run:*

1. pip install fastapi uvicorn

2. uvicorn main:app --reload

3. Visit http://127.0.0.1:9000/docs
